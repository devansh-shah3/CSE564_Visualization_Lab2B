Dataset Information: FIFA22 Dataset.


Link to dataset

https://www.kaggle.com/cashncarry/fifa-22-complete-player-dataset

About Data

Data for this assignment(Lab2) is obtained from Kaggle. It contains all the data about all the fifa 22 player's attributes and information.

Attributes

  1. Name: Players Name (categorical variable)

  2. Age: Player's Age

  3. Height: Player's Height in cms

  4. Overall: Player's Overall Rating in FIFA 22

  5. BestPosition: Player's Best Position in FIFA 22 (categorical variable)

  6. SkillMoves: Player's SKill Moves (5 star, 4 star, 3 star, 2 star, 1 star) Rating (categorical variable)

  7. PaceTotal: Speed at which the player can run

  8. ShootingTotal: Shooting attribute score

  9. PassingTotal: Passing attribute score

  10. DribblingTotal: Dribbling attribute score

  11. DefendingTotal: Defending attribute score

  12. Jumping: Jumping attribute score

  13. Finishing: Finishing attribute score

  14. Vision: Player's Vision Capability

  15. BallControl: Player's Ball Control Capability

  16. FKAccuracy: Free kick accuracy


Intrinsic Dimensionality Index : None
